== Description ==

The "Update Icon for Twitter X" plugin helps keep your website's Twitter icon fresh and modern by automatically updating it to the latest X icon. This plugin checks if the Font Awesome library is installed and activated. If not, it guides you through the installation process seamlessly. Once set up, it injects custom CSS to give your Twitter icon a contemporary look, aligning with the latest design trends. Perfect for maintaining a sharp and up-to-date appearance on your site with minimal effort!

== Installation ==

1. Upload the `update-twitter-x-icon` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure that the Font Awesome plugin is installed and activated. If not, the plugin will prompt you to do so.
4. Once activated, the plugin will automatically update the Twitter icon across your site.

== Frequently Asked Questions ==

= What does this plugin do? =

This plugin updates the Twitter icon on your website to the latest X icon using the Font Awesome library.

= Do I need to have Font Awesome installed? =

Yes, this plugin requires Font Awesome to be installed and activated. If it's not installed, the plugin will prompt you to install and activate it via a provided link.

= How do I install Font Awesome? =

You can install Font Awesome from the WordPress plugin repository. The "Update Icon for Twitter X" plugin will provide a link and guide you through the installation process if Font Awesome is not detected.

= How can I support the development of this plugin? =

If you find this plugin useful, please consider making a donation to support its development. A donation link is provided in the WordPress admin dashboard when the plugin is activated.
